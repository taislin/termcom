## 概要

**termcom** は X-COM: UFO Defense (1994) のローグライク化された ASCII デメイクで、
完全に端末上に描画されます。あなたは X-COM — 地球をエイリアンの侵略から守る
国際任務部隊 — を指揮します。

**目的:** エイリアン技術を研究し、武器と防具を製造し、小隊を率いて戦術戦闘で
エイリアンの脅威を排除することです。

**勝利:** 十分な数の戦闘に勝利してサイドニア最終ミッションを発生させ、それを達成する。

**敗北:** エイリアン活動が 100% に達する — 侵略が地球を飲み込む。

**難易度:** 開始前にレベルを選択します。高い難易度ほどエイリアンが強く、
UFO が頻繁に出現し、初期資金が厳しくなります。

- **初心者** — 弱いエイリアン、遅い UFO、多めの初期資金
- **経験者** — 標準
- **ベテラン** — 強いエイリアン、速い UFO、少ない資金
- **天才** — 全般的にかなり難しい
- **超人** — 最大のエイリアンの脅威

**言語:** 8 言語に対応 — オプション画面で切り替え可能。
英語、中国語、スペイン語、フランス語、ロシア語、ポルトガル語、日本語、韓国語。

**オプション:** 任意の画面で `?` を押してヘルプを開くか、オプション画面に移動して
ブルーム、ライティング、サウンド、オートセーブ、画面振動、マウスサポート、
グリッド線、確認ダイアログ、テーマ、解像度速度、音量、言語を調整します。

## チュートリアル / オンボーディング

最初のプレイ（セーブファイルが検出されない場合）、難易度を選択した直後に
段階的な指揮官**ブリーフィング**が自動的に表示されます。内容は以下の通りです。

1. **ようこそ** — X-COM の紹介
2. **ジオスケープと時間** — 一時停止 (`Space`)、速度 (`1`–`4`)
3. **UFO 検知** — レーダーと UFO マーカー
4. **攻撃機発進** — `L` を押して迎撃
5. **ミッション対応** — `M` を押して展開
6. **基地管理** — `B` を押して基地を管理
7. **バトルスケープ** — 行動ポイント、移動、戦闘
8. **完了** — 準備完了

**操作:** Enter で進む、S でスキップ、Esc で閉じる。

**再生:** オプション画面を開き、いつでも「チュートリアルを再生」を選択します。

## はじめに

あなたは **ジオスケープ** — 世界地図 — から始まります。時間は自動的に進みます。
UFO はレーダー圏内に入るとマーカーとして表示されます。

**初期リソース:**
- $500,000 (難易度で修正)
- 科学者 10 名、技師 10 名
- 居住区、研究所、工房、倉庫、レーダーを備えた基地
- ライフルとピストルを数丁

**UFO を検知したとき:**

| 行動 | キー | 内容 |
|--------|-----|-------------|
| 攻撃機を発進 | `L` | 戦闘機を送り撃墜する |
| 輸送機を派遣 | `R` | 部隊を送り墜落現場を調査する |
| 自動解決 | `A` | 素早い自動迎撃結果 |
| ミッション対応 | `M` | エイリアンのテラー/補給ミッションに展開 |

## ジオスケープ

ジオスケープは各地域の脅威レベルを表示する**地域ダッシュボード**です。

- **左ペイン:** 脅威バーとレーダー状態付きの地域リスト
- **右ペイン:** 基地、UFO、攻撃機、経路を表示する ASCII ミニマップ

### ミニマップ記号

| 記号 | 意味 |
|--------|---------|
| ◆ | あなたの基地 |
| ◉ | 現在選択中のノード |
| ○ | 地域ハブ (緑=安全、黄=脅威、赤=危険) |
| · | レーダー圏内リング |
| ! | UFO (赤、太字) |
| > | 巡回中の攻撃機 |
| ► | UFO と交戦中の攻撃機 |
| ✕ | 撃墜された攻撃機または UFO |
| * | 墜落現場 (黄=未回収、灰=回収済) |
| ≈ | 輸送中の輸送機 (緑) |

### 時間操作

| キー | 速度 | 用途 |
|-----|-------|-----|
| Space | 一時停止 | 計画のため時間を止める |
| 1 | 1x | ゆっくり進行 |
| 2 | 5x | 通常の巡回速度 |
| 3 | 20x | 早送り |
| 4 | 60x | 最大速度 |

### 月次予算

- **収入:** 基本 $200,000 + レーダー施設 1 つにつき $50,000
- **支出:** 兵士・科学者・技師 1 名につき $2,000

### 複数基地

空のノードで `N` を押して新しい基地を建設 ($500K)。各基地は独自の施設・兵士・
倉庫を持ちます。`C` を押して活動中の基地を切り替えます。
`T` を押して転送画面を開き、基地間で兵士やアイテムを移動します。

### ミッション対応

ミッションが発生したとき（テラー、補給襲撃、拉致など）、`M` を押して対応します。

| オプション | 結果 |
|--------|--------|
| **小隊を展開** | 本格的な戦術戦闘 — 報酬最大、リスク最高 |
| **自動解決** | 素早い結果 — 経験値減、死体なし、だが安全 |
| **無視** | スキップ — エイリアン活動が上昇 |

自動解決は実戦の約半分の経験値で、エイリアンの死体はなく、
敗北時にわずかな負傷者の可能性があります。

### 基地防衛

ミッションがあなたの基地のあるノードを標的にした場合、対応すると**基地防衛**
戦闘が始まります。基地防衛に敗北すると、その基地と要員が破壊されます。
最後の基地を失うとゲーム終了です。

### UFO 迎撃

`L` を押して最寄りの UFO に攻撃機を発進させます。攻撃機は追跡し、
短い自動解決のドッグファイトで交戦します。ミニマップには HP バーと
命中/回避のフィードバック付きで交戦が表示されます。

### エイリアンのミッション

ミッションは約 30 ゲーム分ごとに発生し、12〜36 時間のタイマーがあります。

| ミッション | タイマー | 想定される内容 |
|---------|-------|----------------|
| テラー | 24h | 都市マップ、多くの民間人が危険に |
| 補給襲撃 | 24h | UFO 内部、合金/エレリウムのボーナス |
| 拉致 | 24h | 農村マップ、民間人を救出 |
| エイリアン研究 | 24h | UFO 内部、エイリアン技術のボーナス |
| 協会 | 36h | 都市マップ、ボーナス資金 $100K |
| エイリアン基地強襲 | 12h | 岩だらけのエイリアン基地、大量の技術獲得 |

ミッションを放置して期限を過ぎると、エイリアン活動が 10% 上昇します。

## 研究と製造

### 研究

研究タブから、科学者をトピックに配置します。研究はゲーム時間の経過とともに
自動的に進みます。

技術ツリーは**手続き生成**されます — 各プレイごとにシード付きアルゴリズムから
一意のツリーが生成されます。コア技術 (レーザー兵器、パーソナルアーマー、
プラズマ兵器) は常に存在しますが、前提条件とコストは異なります。

**優先順位:**
- **エイリアン合金** と **エレリウム-115** を最初に研究すべき
- エイリアン種の検死は Lore を解禁し、武器技術のゲートになる場合がある
- 捕獲したエイリアンを尋問 (`I` キー) すると研究がより早く完了する

### 製造

製造タブから、技師を配置してアイテムを生産します。
これは**標準の武器と防具**を生産します — より強力なカスタム装備については、
武器設計室と航空機設計室を使用します。

**製造可能なアイテム** (一度に複数製造):

| アイテム | 時間 | 材料 |
|------|------|-----------|
| ピストル、ライフル、ヘビーキャノン、オートキャノン | 3〜8日 | 合金 |
| ロケットランチャー、スタンロッド | 2〜8日 | 合金 + エレリウム |
| パーソナルアーマー、ライト/ミディアム/ヘビー/パワー/フライトスーツ | 6〜18日 | 合金 + エレリウム |
| メディキット | 3日 | 合金 |

技師が多いほど生産が速くなります。エネルギー武器 (レーザー、プラズマ) は
製造できません — 研究し、設計し、武器設計室で作るか、エイリアンから回収する
必要があります。

## 兵士の装備

兵士タブから `E` を押して装備画面を開きます。

### 操作

| キー | 行動 |
|-----|--------|
| ↑/↓ | 兵士を選択 |
| Tab | 利用可能なアイテムを切替 |
| 1 | 武器スロット |
| 2 | 防具スロット |
| 3 | インベントリスロット |
| Space | 選択したアイテムを装備 |
| G | 武器設計室を開く |
| A | 全兵士を自動装備 |
| Esc | 戻る |

### スロット

- **スロット 1 (武器):** 主武器 — カスタム設計、または標準のライフル/ピストル
- **スロット 2 (防具):** 体防具 — パーソナルアーマー、ライトスーツなど
- **スロット 3 (インベントリ):** 追加アイテム — 手榴弾、メディキット、スキャナー、
  接近地雷、サイ増幅器、近接武器

### 重量

すべてのアイテムには重量があります。武器 + 防具 + インベントリの合計重量が
あなたの**重量負荷**です。重量負荷が高いほど、戦闘での行動ポイントが減ります
（おおよそ 5 重量単位につき 1 TU のペナルティ）。最大の機動力を得るため、
兵士は軽装に留めてください。

### 自動装備

`A` を押すと、倉庫から最良の利用可能な武器と防具で全兵士を自動的に装備します。
既存の装備は倉庫に戻ります — 新しい技術を研究した後に小隊を再装備する早い方法です。

## バトルスケープ

バトルスケープはターン制の戦術戦闘です。50×50 グリッドマップ上で、
エイリアン軍に対する兵士の小隊を指揮します。

### ターン構造

1. **プレイヤーターン** — 各行動ポイント (TU) を使って各兵士を移動・行動させる
2. **エイリアンターン** — エイリアンは独自の TU プールを使って行動する
3. 一方が全滅するまで繰り返す

### 行動ポイント (TU)

すべての行動は TU を消費します。TU は各プレイヤーターンの開始時に全回復します。

| 行動 | おおよその TU コスト |
|--------|-----------------|
| 移動 (タイルごと) | 4 |
| 伏せ | 4 |
| 武器を撃つ | 武器により異なる (狙い撃ち=基本、バースト=1.5×、フルオート=2×) |
| リロード | 8 |
| 手榴弾を投げる | 20 |
| メディキットを使う | 25 |
| サイ攻撃 | 20 |

兵士の TU プールは 45〜55 で始まり、経験で成長します (最大 ~80)。

### 射撃モード

武器には複数の射撃モードがある場合があります。**Tab** を押して切り替え、
サイドバーに表示されたモードを確認してください。

| モード | コスト | 精度 | 弾数 | 用途 |
|------|------|----------|--------|-------------|
| **狙い撃ち** | 基本 TU | 最高 | 1 発 | 長距離、高価値目標 |
| **バースト** | 1.5× TU | -10% | 3 発 | 中距離、抑制 |
| **フルオート** | 2× TU | -20% | 残り全弾 | 近距離、緊急時 |

すべての武器がすべてのモードに対応しているわけではありません。ライフルと
レーザーライフルはバーストに対応し、フルオートに対応する武器はごく一部です。

### 戦闘

戦闘の要素:
- **精度** は兵士の技量、目標までの距離、掩蔽、使用する射撃モードに依存
- **伏せ** は精度ボーナスを与え、受けるダメージを軽減する
- **掩蔽** — 壁は 80% のダメージを遮断し、木は 60%、茂みは 40%、
  柵は 30%。兵士を頑丈な掩蔽の後ろに配置すること
- 手榴弾で**掩蔽を無視** — 範囲爆発し、掩蔽のダメージ軽減を無視する

### 視線 (LOS)

兵士は直線上でしか見ることができません。壁・木・岩は視線を遮断します。
床・扉・草は遮断しません。

### 物体と掩蔽

物体を通過する弾は、その物体の掩蔽値に応じてダメージが軽減されます。
射線上で最も高い掩蔽が適用されます。

| 物体 | 掩蔽 % |
|--------|---------|
| 壁 / UFO の壁 | 80% |
| 岩 | 70% |
| 木 | 60% |
| UFO 家具 | 50% |
| 茂み | 40% |
| 濃い煙 | 40% |
| 柵 | 30% |
| 瓦礫 | 20% |

### 手榴弾

- 射程: 約 6 タイル
- ダメージ: 強さに基づき、範囲に飛散
- 爆風半径内の壁・木・岩・柵を破壊する
- 高密度で視線を遮断する煙の雲を生成する

### メディキット

使用ごとに 10 HP 回復 (戦場衛生兵の特技なら 15 HP)、25 TU を消費します。

### 夜間ミッション

夜間 (6:00 前または 18:00 後):
- 精度低下 (昼間の約 75%)
- 視界範囲短縮 (20 から約 10 タイルへ)
- 兵士は暖かく光り、エイリアンは淡い青に光る

### 視界モード

`V` を押して切り替: **通常 → 暗視 → 熱感知 → 通常**
- 暗視: 静的のある緑色の蛍光オーバーレイ
- 熱感知: 生きている実体は熱く光り、地形は冷たい青

### サイ戦闘

サイ増幅器武器とサイ研究所施設が必要です。成功はあなたの兵士のサイ能力と
目標のサイ強度の差に依存します。サイ攻撃が成功すると、目標はパニックし —
ターンを失います。

サイ研究所のある基地の兵士は、時間とともにサイ能力を獲得する場合があります (~80 まで)。
精神制御の研究は全兵士に大幅なサイブーストを与えます。

### ミッションモディファイア

各戦闘ごとに変化するランダムなモディファイア:

| モディファイア | 内容 |
|----------|--------------|
| 夜間作戦 | 強制夜間戦闘、ボーナス戦利品 |
| 増援 | ターン 4 に追加のエイリアンが到着 |
| 制限時間 | 全エイリアン排除まで 15 ターン |
| VIP 救出 | VIP を保護、生存でボーナス資金 |
| ブービートラップ | マップ上の手榴弾と地雷が増加 |
| 濃霧 | 視界範囲が 40% 減少 |
| エイリアンの待ち伏せ | エイリアンがオーバーウォッチ位置で開始 |
| 低視界 | 全ユニットの精度低下 |
| 高所 | 高所で精度ボーナス |

### 天気

天気はミッションの場所に基づいて戦闘に影響します。

| 天気 | 効果 |
|---------|--------|
| 雨 | 精度低下、火の広がりが遅い |
| 風 | 火の広がりが速い、手榴弾が流される場合がある |
| 雪 | 深い雪での移動コスト増 |
| 霧 | 精度低下、視界減 |
| 嵐 | 雨 + 風の組み合わせ |
| 寒い | わずかな精度ペナルティ |

### 事後報告書

各戦闘の後、以下が表示されます。
- 結果 (勝利 / 敗北)
- 排除したエイリアンと失った兵士
- 回収した戦利品と捕獲した捕虜
- 獲得した資金
- 兵士ごとのステータス上昇または「戦死」マーク

**Enter**、**Space**、または **Esc** を押して閉じます。

## 武器と装備

### カスタム武器設計室

基地、兵士、または装備画面から `G` を押して**武器設計室**を開きます。
これは兵士用の武器を作る主要な方法です。基本テンプレートを選び、
各パーツをカスタマイズします。

| パーツ | オプション | 影響 |
|-----------|---------|-----------------|
| **ベース** | ピストル / ライフル | 初期ダメージ、射程、精度、TU コスト |
| **銃身** | 短銃身 / 標準銃身 / 長銃身 / 延長銃身 | 射程、精度、TU コスト、重量 |
| **光学** | なし / アイアンサイト / スコープ / 先進光学機器 | 精度、TU コスト、重量 |
| **射撃モード** | セミ / フル | フルオートモード (より速く発射、精度は低い) |
| **弾薬** | 標準 / 徹甲 / 焼夷 / 炸裂 | ダメージ修正、TU コスト、重量 |
| **銃床** | なし / 軽量ストック / 重量ストック | 精度、TU コスト、重量 |

各パーツは武器のダメージ、精度、TU コスト、射程、重量に影響します。
プレビューパネルには組み立てられた武器が色付き ASCII アートで表示され、
最終ステータスが示されます。設計は装備画面で利用可能なカスタムアイテムとして
保存されます。

**ヒント:** 多くの用途でライフルベースから始めましょう。長銃身とスコープは
遠距離精度を向上させます。炸裂弾薬は強力ですが、TU を余分に消費します。

|------|--------|-------|

### 弾薬とリロード

- **実弾武器** はリロードが必要 — 戦闘中に `R` を押す
- **エネルギー武器** (レーザー、プラズマ) はリロード不要
- **消耗品** (手榴弾、メディキット) はインベントリから使用

### 射撃モード

詳細は [バトルスケープ → 射撃モード](#射撃モード) を参照。

### インベントリアイテム

兵士はインベントリスロットに追加アイテムを携帯できます。
- **手榴弾** — 範囲ダメージのある投擲爆発物
- **メディキット** — 自分または隣接する味方を回復
- **モーションスキャナー** — 近くの敵を検出
- **接近地雷** — 地面に設置、敵が通ると起爆
- **サイ増幅器** — サイ攻撃を可能に (サイ能力が必要)
- **近接武器** — 非致死な制圧用のスタンロッド

各インベントリアイテムは重量を加え、重量負荷を増やし、戦闘での利用可能な
TU を減らします。賢明に詰め込んでください。

### 手続き生成アイテム

各プレイでは、遭遇したエイリアン種に基づいた一意の武器と防具が生成されます。
これらはランダムな名前とステータスを持ちます — ゲームごとに異なります。

**手続き生成武器:** エイリアン種のダメージタイプに合う 2〜3 の武器。
**手続き生成防具:** エイリアンのダメージタイプに合う保護を持つ 1〜2 の防具。

これらのアイテムはゲーム開始時に自動的に倉庫に追加されます。

## エイリアン

### 手続き生成種

各ゲームではシードから 5〜7 の一意のエイリアン種が生成されます。
各種には 2〜5 の階級バリアントがあります (兵士 → 航海士 → 指揮官 → エリート → 覇王)。

種によって異なる点:
- **ダメージタイプ** — 与えるダメージの種類
- **耐性と弱点** — プラズマに弱いもの、爆発に弱いものなど
- **武器の好み** — 低階級はピストル、高階級は重武器を使う
- **形態** — ステータスと耐性に影響する身体的な体計画

つまり、**各プレイは異なるエイリアンの脅威を特徴とします**。ある周回は
サイ能力重視の種かもしれず、別の周回は爆発に弱い近接の捕食者かもしれません。

### 形態

形態はエイリアンの物理的形態を決定します。主要な要素:

**肢:**
- 腕 (0〜6): 腕が少ない = 精度が悪い、腕が多い = 安定性または二刀流が良い
- 脚 (0〜8): 脚が多い = 速いが大きな標的、脚 0 = 浮遊、命中させにくい

**体タイプとその耐性:**
- **肉と骨:** +運動エネルギー耐性、-爆発弱点
- **ケイ素系結晶組織:** +レーザー/+プラズマ耐性、-爆発弱点、反射
- **半実体の気体状態:** 運動エネルギーに免疫、プラズマに弱い、壁をすり抜ける
- **緻密な鉱物結晶構造:** 全般的な耐性が良い、爆発に非常に弱い、死亡時に砕ける
- **常に変化する不定形の体:** +サイ耐性、毎ターン HP を再生
- **精密鍛造の機械部品:** サイに免疫、+プラズマ耐性、-レーザー弱点、自爆
- **有機組織と合成装甲の融合体:** バランスの取れた耐性、隣接するエイリアンを回復
- **ナノスケール機械の群れ:** +運動エネルギー耐性、死亡時に蘇生可能

**感覚:**
- **目:** 精度に影響 — マルチスペクトルは煙/暗闇を無視
- **聴覚:** 反響定位は近距離で煙を通してユニットを検出
- **熱感知:** 近距離で掩蔽に関係なく生きているユニットを検出
- **サイ感知:** サイを強化、精神制御された人間を検出
- **化学感知:** 負傷した目標へのボーナス精度

### 知識レベル

エイリアンと遭遇するにつれて、情報が向上します。

| レベル | 学ぶこと |
|-------|----------------|
| 不明 | 名前は "???" と表示 |
| 発見 | 名前とアイコンが判明 |
| 排除 | ステータスと耐性が判明 |
| 検死 | 完全な Lore と詳細な弱点 |

### エイリアン AI

エイリアンは人間を発見するまで巡回し、その後攻撃します。行動には以下が含まれます。
- **索敵** — 数ターン、最後に知られた位置へ移動
- **退却** — 重傷で勇気が低いとき逃走
- **適応** — エイリアンはミッションを通してあなたの戦術を学習。
  遠距離から狙撃？ 彼らは突撃する。手榴弾を使う？ 彼らは散開する。
  しばしば側面攻撃？ 彼らは抑制部隊を配置する。

### 装備の上昇

キャンペーンが進むにつれてエイリアンはより良い装備を得ます。
- **初期の月:** プラズマピストル、基本防具
- **中盤:** プラズマライフル、重プラズマ、エイリアンキャノン
- **終盤:** 最上位のエイリアン武器と防具

### エイリアンの捕獲

**スタンロッド** (近接、製造費 $2K) を使ってエイリアンを気絶させます。
気絶ダメージが HP を超えると、彼らは意識を失い、ミッション後に
（空き容量のあるエイリアン収容施設があれば）回収できます。

捕獲したエイリアンは研究画面 (`I` キー) から尋問できます。
- 尋問は進行中の検死を即座に完了できる
- または現在の研究への進捗ボーナスを与える
- 最低 1 つの研究所が必要

## 兵士の階級と成長

### 階級

階級は総名簿が増えるにつれて解放されます。

| 階級 | 名簿が到達したときに解放 |
|------|----------------------------|
| 新兵 | 常に利用可能 |
| 伍長 | 常に利用可能 |
| 軍曹 | 4 名の兵士 |
| 曹長 | 8 名の兵士 |
| 中尉 | 14 名の兵士 |
| 大尉 | 22 名の兵士 |
| 少佐 | 30 名の兵士 |
| 大佐 | 40 名の兵士 |

### ステータス成長

兵士は戦闘中の**行動ごとの経験**を通じて向上します。
- **射撃** → 精度を向上
- **反応** → 反応を向上
- **近接** → 筋力を向上
- **勇気** → 勇気を向上 (パニックへの抵抗から)
- **サイ能力** → サイ能力とサイ強度を向上

各ミッション後、蓄積した経験値はステータス上昇に変換されます。経験値を得た
兵士は、HP・TU・筋力の上限に向けて全般的な「ハロー」成長も得ます。
上限はおおよそ: TU 80、HP 60、精度 120、反応 100、勇気 100、筋力 70、サイ 100。

### 疲労と負傷

- **負傷した兵士** は回復するまで展開できません (2 HP/日の回復)
- **疲労:** 戦闘は 1〜5 日の疲労を引き起こす
- 回復施設と居住区は回復を速めます

### 致命的負傷

戦闘中、命中は致命的負傷と出血を引き起こす場合があります。出血は毎ターン
HP を奪います — 早くメディキットを当ててください。生き残った負傷はミッション後に
回復日数となります。

### 士気

兵士は毎ターン士気を回復します。低い士気はパニック (ターンをスキップ) を
引き起こす可能性があります。パニックへの抵抗は勇気の経験値を築きます。

### 特技

各階級アップでランダムな特技を得ます。

| 特技 | 効果 |
|------|--------|
| 稲妻の反射神経 | +10 反応 |
| 名射手 | 長距離で+精度 |
| 擲弾兵 | 手榴弾の散弾範囲が大きく |
| 戦場衛生兵 | メディキットの回復が増加 |
| 鉄の意志 | +サイ能力 と +サイ強度 |
| 安定した照準 | 静止時に+精度 |
| 接近戦専門家 | 近距離で+精度 |
| 監視専門家 | +反応射撃精度 |
| 破壊工作 | +手榴弾ダメージ |
| 戦利品収集者 | +戦闘での戦利品 |
| 強靭 | +5 最大 HP |
| 素早い学習者 | +経験値獲得 |

### 記念碑

戦死した兵士はゲーム内の記念碑に記録されます。
亡くなった者を称えるために表示できます。

## ヒントと戦略

### 序盤

1. まず **エイリアン合金** を研究 — レーザー兵器と防具を解禁する。
2. 2 つ目の **レーダー** を建設 — 検知が増え、月次資金が増える。
3. 2〜4 名の追加兵士を雇用して小隊を埋める。
4. **武器設計室** を使ってカスタムライフルを作る — 正しいパーツで標準モデルより
   良い銃が作れる。
5. 検死を無視しない — 一部の武器技術はそれを必要とする。
6. **航空機設計室** で攻撃機をカスタム設計 — バランスの取れた設計
   (中程度の全長/翼幅、エンジン 2、スティングレイミサイル) はデフォルトの
   攻撃機を上回る。

### 戦闘

- **掩蔽を使う** — 壁 (80%) > 岩 (70%) > 木 (60%) > 茂み (40%)
- **射撃前に伏せる** — より良い精度とダメージ軽減のため
- **手榴弾** は掩蔽を無視し壁を破壊 — 塹壕に篭る敵に最適
- **エイリアンの耐性を学ぶ** — 最初の排除後に大百科事典を確認
- **突出しすぎない** — 敵の視線内を移動すると反応射撃を受ける
- **衛生兵を確保** — メディキットを持つ 1 人の兵士が命を救う
- **重量負荷を管理** — 兵士に重い装備を過積載しない

### 経済

- 余ったエイリアンの死体と戦利品を現金に売却
- 月次給与は積み重なる — 収入に対して名簿をバランスさせる
- 協会ミッションは $100K のボーナス — 優先する
- 序盤はアイテムを製造して販売益を出す

### 研究の道筋

合金 → レーザー兵器 → パーソナルアーマー → 検死 → エレリウム → プラズマ兵器

中盤: ミディアムスーツ、重プラズマ。
終盤: パワー/フライトスーツ、精神制御。
