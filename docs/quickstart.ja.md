# termcom クイックスタート

あなたの端末のための ASCII X-COM デメイク。エイリアン侵略に対する人類の防衛を指揮します。

## 実行

```bash
go run ./cmd/termcom      # または: make run
```

## ゲームループ

1. **ジオスケープ** -- UFO が都市へ向かって飛ぶ。検知して迎撃する。
2. **迎撃** -- 戦闘機を発進 (L) または自動解決 (A) して UFO を撃墜する。
3. **戦闘** -- 墜落現場へ展開 (R)。戦術戦闘に入る。
4. **基地** -- エイリアン技術を研究、装備を製造、兵士を雇用/装備する。
5. **繰り返す** -- 10 回の戦闘に勝利し、その後サイドニアを強襲して地球を救う。

エイリアン活動が 100% に達すると敗北。

## 基本キー (ジオスケープ)

| キー | 行動 |
|-----|--------|
| Space | 一時停止 |
| 1-4 | 時間速度 |
| L | 攻撃機発進 |
| A | UFO を自動解決 |
| M | ミッションに対応 |
| R | 墜落現場へ輸送機を送る |
| B | 基地を開く |
| F5/F9 | セーブ/ロード |
| Q | 終了 |

## 基本キー (バトルスケープ)

| キー | 行動 |
|-----|--------|
| 矢印/WASD | カーソル移動 |
| Space/Enter | 選択/確定 |
| F | 武器を撃つ |
| R | リロード |
| Q | 兵士を切替 |
| E | ターン終了 |
| C | 伏せ |
| Esc | キャンセル |

## クイック戦略

- **序盤:** 兵士を雇用、エイリアン合金を研究、研究所 + 工房を建設
- **中盤:** カスタムレーザー武器 (武器設計室) → パーソナルアーマー、基地を拡張
- **終盤:** カスタムプラズマ武器、パワー/フライトスーツ、サイ訓練
- 戦闘前に必ず兵士を装備させる。負傷は 2 HP/日で回復。
- 航空機設計室でカスタム攻撃機を設計 — スティングレイミサイル + 軽量合金装甲が堅実な出発点。
- 余ったエイリアンのアーティファクトを現金に売却。レーダー施設は資金を増やす。

## 勝利

地上戦を 10 回勝利するとサイドニア最終ミッションが解禁されます。サイドニアを破壊して勝利。
