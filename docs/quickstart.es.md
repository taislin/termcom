# Inicio rápido de termcom

Una demake de X-COM en ASCII para vuestra terminal. Comandad la defensa de la humanidad contra la invasión alienígena.

## Ejecutar

```bash
go run ./cmd/termcom      # o: make run
```

## Bucle de jugabilidad

1. **Geoscape** -- Los OVNIs vuelan hacia las ciudades. Detectadlos e interceptadlos.
2. **Interceptar** -- Lanzad cazas (L) o autoresolved (A) para derribar OVNIs.
3. **Batalla** -- Desplegad en puntos de impacto (R). Entrad en combate táctico.
4. **Base** -- Investigad tecnología alienígena, fabricad equipo, contratad/equipad soldados.
5. **Repetir** -- Ganad 10 batallas, luego asaltad Cydonia para salvar la Tierra.

Perded si la Actividad Alienígena alcanza el 100%.

## Teclas esenciales (Geoscape)

| Tecla | Acción |
|-------|--------|
| Space | Pausa |
| 1-4 | Velocidad del tiempo |
| L | Lanzar interceptor |
| A | Autoresolver OVNI |
| M | Responder a misión |
| R | Enviar transporte a impacto |
| B | Abrir base |
| F5/F9 | Guardar/Cargar |
| Q | Salir |

## Teclas esenciales (Campo de Batalla)

| Tecla | Acción |
|-------|--------|
| Flecha/WASD | Mover cursor |
| Space/Enter | Seleccionar/Confirmar |
| F | Disparar arma |
| R | Recargar |
| Q | Ciclar soldado |
| E | Terminar turno |
| C | Agacharse |
| Esc | Cancelar |

## Estrategia rápida

- **Principios:** Contratad soldados, investigad Aleaciones Alienígenas, construid Laboratorio + Taller
- **Mitad:** Armas láser personalizadas (Diseñador de Armas) → Armadura Personal, expandid bases
- **Final:** Armas de plasma personalizadas, Trajes de Potencia/Volador, entrenamiento psi
- Equipad siempre a los soldados antes de la batalla. Los heridos curan 2 HP/día.
- Diseñad interceptores personalizados en el Diseñador de Aviones — misiles Stingray + blindaje de Aleación ligera es un buen comienzo.
- Vended artefactos alienígenas sobrantes por dinero. Las instalaciones de Radar suben la financiación.

## Victoria

Ganad 10 batallas terrestres para desbloquear la misión final de Cydonia. Destruid Cydonia para ganar.
