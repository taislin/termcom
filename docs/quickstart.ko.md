# termcom 빠른 시작

터미널을 위한 ASCII X-COM 리메이크. 외계 침공에 맞서 인류의 방어를 지휘하십시오.

## 실행

### 터미널 버전

```bash
go run ./cmd/termcom      # 또는: make run
```

### 브라우저 버전 (WASM)

백엔드 서버 불필요 — 브라우저에서 직접 실행.

```bash
# 빌드 및 서빙
cd cmd/termcom_wasm
GOOS=js GOARCH=wasm go build -o ../../web_wasm/termcom.wasm .
cd ../../web_wasm
python -m http.server 8080
# http://localhost:8080 열기
```

또는 빌드 스크립트 사용: `./scripts/build_wasm.sh`

## 게임 진행 루프

1. **지구전략맵** -- UFO가 도시를 향해 비행합니다. 탐지하고 요격하십시오.
2. **요격** -- 전투기 발진 (L) 또는 자동해결 (A)로 UFO를 격추.
3. **전투** -- 추락 지점에 투입 (R). 전술 전투 진입.
4. **기지** -- 외계 기술 연구, 장비 제조, 병사 고용/장비.
5. **반복** -- 전투 10번 승리한 뒤 지구를 구하기 위해 사이도니아 공격.

외계인 활동도가 100%에 도달하면 패배.

## 필수 키 (지구전략맵)

| 키 | 동작 |
|-----|--------|
| Space | 일시정지 |
| 1-4 | 시간 속도 |
| L | 요격기 발진 |
| A | UFO 자동해결 |
| M | 미션 대응 |
| R | 추락 지점으로 수송 파견 |
| B | 기지 열기 |
| F5/F9 | 저장/불러오기 |
| Q | 종료 |

## 필수 키 (전술전장)

| 키 | 동작 |
|-----|--------|
| 화살표/WASD | 커서 이동 |
| Space/Enter | 선택/확인 |
| F | 무기 발사 |
| R | 재장전 |
| Q | 병사 순환 |
| E | 턴 종료 |
| C | 엎드리기 |
| Esc | 취소 |

## 빠른 전략

- **초반:** 병사 고용, 외계 합금 연구, 연구소 + 공방 건설
- **중반:** 맞춤 레이저 무기 (무기 설계기) → 개인용 갑옷, 기지 확장
- **후반:** 맞춤 플라즈마 무기, 동력/비행 슈트, 초감각 훈련
- 전투 전에는 항상 병사를 장비하십시오. 부상은 2 HP/일 회복.
- 비행기 설계기에서 맞춤 요격기를 설계하십시오 — 스팅레이 미사일 + 경합금 장갑이 탄탄한 시작입니다.
- 남는 외계 유물을 팔아 현금을 얻으십시오. 레이더 시설은 자금을 늘립니다.

## 승리

지상 전투 10번에서 승리해 사이도니아 최종 미션을 해금하십시오. 사이도니아를 파괴하면 승리.
