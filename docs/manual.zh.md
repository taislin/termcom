## 概览

**termcom** 是 X-COM: UFO Defense（1994）的 roguelike 化 ASCII 重制版，完全在终端中
以彩色 ASCII 渲染。您指挥 X-COM —— 一支保卫地球、抵御外星入侵的国际特遣队。

**您的目标：** 研究外星科技，制造武器与护甲，并率领小队投入战术战斗以消灭
外星威胁。

**胜利：** 赢得足够多的战斗以触发赛多尼亚最终任务，然后将其攻下。

**失败：** 外星活动达到 100% —— 入侵彻底压垮地球。

**难度：** 在开始前应选择一个难度。难度越高，异形越强，UFO 越频繁，起始资金越紧张。

- **新手** —— 异形较弱，UFO 较慢，起始资金更多
- **经验丰富** —— 标准
- **老兵** —— 异形更强，UFO 更快，资金更少
- **天才** —— 各方面都困难得多
- **超人** —— 最大异形威胁

**语言：** 提供 8 种语言 —— 在选项界面中切换。
英语、中文、西班牙语、法语、俄语、葡萄牙语、日语、韩语。

**选项：** 在任意界面按 `?` 打开帮助，或进入选项界面以调整发光效果、光照、音效、
自动存档、屏幕震动、鼠标支持、网格线、确认对话框、主题、分辨率速度、音量和语言。

## 教程 / 新手引导

在您的首次游玩中（未检测到存档文件时），选择难度后会自动出现逐步的指挥官
**简报**。其内容包括：

1. **欢迎** —— X-COM 简介
2. **地球仪与时间** —— 暂停（`Space`）、速度（`1`–`4`）
3. **UFO 探测** —— 雷达与 UFO 标记
4. **截击机发射** —— 按 `L` 交战
5. **任务响应** —— 按 `M` 部署
6. **基地管理** —— 按 `B` 管理您的基地
7. **战场画面** —— 行动点、移动与战斗
8. **完成** —— 您已准备就绪

**操作：** 回车前进，S 跳过，Esc 关闭。

**重播：** 随时打开选项界面并选择“重播教程”。

## 入门指南

您从**地球仪表盘** —— 世界地图 —— 开始。时间自动推进。
UFO 进入雷达范围后会出现在地图上。

**起始资源：**
- $500,000（随难度调整）
- 10 名科学家、10 名工程师
- 一座拥有宿舍、实验室、车间、仓库和雷达的基地
- 数支步枪与手枪

**当探测到 UFO 时：**

| 操作 | 按键 | 发生的事情 |
|--------|-----|-------------|
| 发射截击机 | `L` | 派出战斗机将其击落 |
| 派遣运输机 | `R` | 派遣部队调查坠毁地点 |
| 自动决斗 | `A` | 快速自动拦截结果 |
| 响应任务 | `M` | 部署应对异形恐怖/补给任务 |

## 地球仪表盘

地球仪表盘显示一个**区域仪表板**，列出各区域的威胁等级：

- **左侧面板：** 区域列表，含威胁条与雷达状态
- **右侧面板：** ASCII 小地图，显示基地、UFO、截击机与航线

### 小地图符号

| 符号 | 含义 |
|--------|---------|
| ◆ | 您的基地 |
| ◉ | 当前选中的节点 |
| ○ | 区域枢纽（绿色=安全，黄色=威胁，红色=危险） |
| · | 雷达覆盖环 |
| ! | UFO（红色，加粗） |
| > | 截击机巡逻中 |
| ► | 截击机正在交战 UFO |
| ✕ | 被摧毁的截击机或 UFO |
| * | 坠毁地点（黄色=未回收，灰色=已回收） |
| ≈ | 运输机在途中（绿色） |

### 时间控制

| 按键 | 速度 | 用途 |
|-----|-------|-----|
| Space | 暂停 | 停止时间以制定计划 |
| 1 | 1x | 缓慢推进 |
| 2 | 5x | 正常巡逻速度 |
| 3 | 20x | 快进 |
| 4 | 60x | 最大速度 |

### 月度预算

- **收入：** $200,000 基础 + 每个雷达设施 $50,000
- **支出：** 每名士兵、科学家和工程师 $2,000

### 多基地

在空节点上按 `N` 建造新基地（$500K）。每个基地拥有自己的设施、士兵与仓库。
按 `C` 切换活动基地。按 `T` 打开转运输界面，在基地间转移士兵或物资。

### 任务响应

当任务出现时（恐怖、补给突袭、绑架等），按 `M` 响应：

| 选项 | 结果 |
|--------|--------|
| **部署小队** | 完整战术战斗 —— 奖励最佳，风险最高 |
| **自动决斗** | 快速结果 —— 经验减少，无尸体，但安全 |
| **忽略** | 跳过它 —— 异形活动上升 |

自动决斗给予约真实战斗一半的经验、没有异形尸体，且失败时有少量伤亡几率。

### 基地防御

如果任务目标是您基地所在的节点，响应将发起一场**基地防御**战斗。
输掉基地防御会摧毁该基地及其人员。失去最后一个基地则游戏结束。

### UFO 拦截

按 `L` 向最近的 UFO 发射截击机。截击机会追击并参与一场短暂自动决斗的空战。
小地图以生命条和命中/未命中反馈显示交战情况。

### 异形任务

任务每约 30 个游戏分钟出现一次，计时为 12–36 小时：

| 任务 | 计时 | 预期内容 |
|---------|-------|----------------|
| 恐怖袭击 | 24h | 城市地图，许多平民处于危险中 |
| 补给袭击 | 24h | UFO 内部，额外合金/埃勒里姆-115 |
| 绑架 | 24h | 乡村地图，营救平民 |
| 外星研究 | 24h | UFO 内部，额外外星科技 |
| 委员会 | 36h | 城市地图，额外 $100K 资金 |
| 外星基地突袭 | 12h | 岩石外星基地，大量科技收获 |

让任务过期会使异形活动增加 10%。

## 研究与制造

### 研究

从研究标签，将科学家分配到课题。研究会随着游戏时间流逝自动推进。

科技树是**程序化**生成的 —— 每次游玩都从带种子的算法生成独特的科技树。
核心技术（激光武器、个人护甲、等离子武器）始终存在，但先决条件与成本各不相同。

**优先级：**
- 应首先研究**外星合金**和**埃勒里姆-115**
- 异形物种的尸体解剖可解锁背景知识，并可能制约武器科技
- 审讯被俘异形（`I` 键）以更快完成研究

### 制造

从制造标签，将工程师分配到物品生产。
这会生产**量产武器与护甲** —— 若要更强力的自定义装备，请使用武器设计室与
飞机设计室。

**可制造物品**（可一次建造多个）：

| 物品 | 时间 | 材料 |
|------|------|-----------|
| 手枪、步枪、重型火炮、自动火炮 | 3–8 天 | 合金 |
| 火箭发射器、电击棒 | 2–8 天 | 合金 + 埃勒里姆 |
| 个人护甲、轻型/中型/重型/动力/飞行服 | 6–18 天 | 合金 + 埃勒里姆 |
| 医疗包 | 3 天 | 合金 |

工程师越多 = 生产越快。能量武器（激光、等离子）无法制造 —— 必须通过
武器设计室研究、设计并建造，或从异形处缴获。

## 装备士兵

从士兵标签，按 `E` 打开装备界面。

### 操作

| 按键 | 操作 |
|-----|--------|
| ↑/↓ | 选择士兵 |
| Tab | 循环可用物品 |
| 1 | 武器槽 |
| 2 | 护甲槽 |
| 3 | 库存槽 |
| Space | 装备所选物品 |
| G | 打开武器设计室 |
| A | 自动装备所有士兵 |
| Esc | 返回 |

### 槽位

- **槽位 1（武器）：** 主武器 —— 自定义设计或量产步枪/手枪
- **槽位 2（护甲）：** 身体护甲 —— 个人、轻型服等
- **槽位 3（库存）：** 额外物品 —— 手榴弹、医疗包、扫描仪、近炸地雷、灵能增幅器、近战武器

### 负重

每件物品都有重量。武器 + 护甲 + 库存的总重量即为您的**负重**。负重越高，
战斗中您的行动点越少（约每 5 重量单位 -1 行动点）。让士兵轻装上阵以保持最高机动性。

### 自动装备

按 `A` 自动为每名士兵装备仓库中最好的可用武器与护甲。
现有装备退回仓库 —— 是研究出新科技后快速重新武装小队的便捷方式。

## 战场画面

战场画面是回合制战术战斗。您指挥一支士兵小队，在 50×50 网格地图上对抗异形部队。

### 回合结构

1. **玩家回合** —— 使用行动点（TU）移动并操作每名士兵
2. **异形回合** —— 异形使用各自的行动点池行动
3. 重复，直到一方被消灭

### 行动点（TU）

每个动作都消耗行动点。行动点在每个玩家回合开始时完全恢复。

| 动作 | 约略行动点消耗 |
|--------|-----------------|
| 移动（每格） | 4 |
| 蹲下 | 4 |
| 开火 | 随武器而异（精准=基础，爆发=1.5×，全自动=2×） |
| 换弹 | 8 |
| 投手榴弹 | 20 |
| 使用医疗包 | 25 |
| 灵能攻击 | 20 |

士兵的行动点池起始为 45–55，并可通过经验成长（最高约 80）。

### 射击模式

武器可有多种射击模式。按 **Tab** 循环，并查看侧栏中显示的模式。

| 模式 | 消耗 | 命中率 | 弹数 | 适用时机 |
|------|------|----------|--------|-------------|
| **精准** | 基础行动点 | 最佳 | 1 发 | 远距离、高价值目标 |
| **爆发** | 1.5× 行动点 | -10% | 3 发 | 中距离、压制 |
| **全自动** | 2× 行动点 | -20% | 所有剩余弹药 | 近距离、紧急情况 |

并非所有武器都支持所有模式。步枪与激光步枪支持爆发；
仅少数武器支持全自动射击。

### 战斗

战斗因素：
- **命中率**取决于士兵技能、距目标距离、掩体与所用射击模式
- **蹲下**提供命中加成并减少受到的伤害
- **掩体** —— 墙阻挡 80% 伤害，树 60%，灌木 40%，栅栏 30%。将士兵布置在坚实掩体后
- 用手榴弹**绕过掩体** —— 它们在范围内爆炸并忽略掩体的伤害减免

### 视线

士兵只能沿直线看见。墙、树和岩石阻挡视线。
地板、门和草不会。

### 物体与掩体

穿过物体的射击，其伤害按物体的掩体值减少。
沿火线中最高的掩体值被应用。

| 物体 | 掩体 % |
|--------|---------|
| 墙 / UFO 墙 | 80% |
| 岩石 | 70% |
| 树 | 60% |
| UFO 家具 | 50% |
| 灌木 | 40% |
| 浓烟 | 40% |
| 栅栏 | 30% |
| 碎石 | 20% |

### 手榴弹

- 射程：约 6 格
- 伤害：基于力量，带范围溅射
- 摧毁爆炸半径内的墙、树、岩石和栅栏
- 产生在高密度时阻挡视线的烟云

### 医疗包

每次使用治疗 10 HP（拥有战地医疗兵专精时为 15 HP），消耗 25 行动点。

### 夜间任务

夜间（6:00 前或 18:00 后）：
- 命中率降低（约为白天的 75%）
- 视距缩短（从 20 降至约 10 格）
- 士兵散发温暖光芒，异形散发微弱蓝光

### 视觉模式

按 `V` 循环：**正常 → 夜视 → 热成像 → 正常**
- 夜视：带噪点的绿色荧光叠加
- 热成像：生物实体发烫发光，地形为冷蓝色

### 灵能战斗

需要灵能增幅器武器与灵能实验室设施。成功取决于您士兵的灵能技能与目标的灵能力量。
成功的灵能攻击使目标恐慌 —— 他们失去回合。

在有灵能实验室的基地中的士兵，可随时间获得灵能技能（最高约 80）。
精神控制研究为所有士兵带来显著的灵能提升。

### 任务修改器

改变每场战斗的随机修改器：

| 修改器 | 发生的事情 |
|----------|--------------|
| 夜间行动 | 强制夜战，奖励战利品 |
| 援军 | 第 4 回合有额外异形到达 |
| 时间限制 | 15 回合内消灭所有异形 |
| VIP 营救 | 保护 VIP，其存活则奖励资金 |
| 布满陷阱 | 地图上有更多手榴弹和地雷 |
| 浓雾 | 视距降低 40% |
| 异形埋伏 | 异形从警戒位置开始 |
| 低能见度 | 所有单位命中率降低 |
| 高地 | 高地位置提供命中加成 |

### 天气

天气根据任务地点影响战斗：

| 天气 | 效果 |
|---------|--------|
| 雨 | 命中率降低，火势蔓延更慢 |
| 风 | 火势蔓延更快，手榴弹可能漂移 |
| 雪 | 在深雪中移动消耗更多 |
| 雾 | 命中率降低，视距缩短 |
| 风暴 | 雨 + 风结合 |
| 寒冷 | 轻微命中惩罚 |

### 战后报告

每场战斗后，您会看到：
- 结果（胜利 / 失败）
- 击杀的异形与损失的士兵
- 回收的战利品与俘虏
- 赚取的资金
- 每名士兵的属性增益或“阵亡”标记

按 **Enter**、**Space** 或 **Esc** 关闭。

## 武器与装备

### 自定义武器设计室

从基地、士兵或装备界面按 `G` 打开**武器设计室**。
这是为士兵创建武器的主要方式。选择基础模板并自定义每个部件：

| 部件 | 选项 | 影响的内容 |
|-----------|---------|-----------------|
| **基础** | 手枪 / 步枪 | 起始伤害、射程、命中率、行动点消耗 |
| **枪管** | 短管 / 标准 / 长管 / 加长 | 射程、命中率、行动点消耗、重量 |
| **光学** | 无 / 机械瞄具 / 瞄准镜 / 先进光学 | 命中率、行动点消耗、重量 |
| **射击模式** | 半自动 / 全自动 | 全自动模式（射速更快，命中率更低） |
| **弹药** | 标准 / 穿甲 / 燃烧 / 爆炸 | 伤害修正、行动点消耗、重量 |
| **枪托** | 无 / 轻型 / 重型 | 命中率、行动点消耗、重量 |

每个部件都会影响武器的伤害、命中率、行动点消耗、射程和重量。
预览面板以彩色 ASCII 艺术显示组装好的武器，并展示其最终属性。设计作为
自定义物品保存，可在装备界面使用。

**提示：** 多数情况下从步枪基础开始。长枪管和瞄准镜提升远程命中率。
爆炸弹药威力大但消耗额外行动点。

### 量产武器

这些基础物品从一开始即可用，且可制造：

## 异形

### 程序化物种

每局游戏从一个种子生成 5–7 个独特的异形物种。每个物种有
2–5 个军衔变体（士兵 → 领航者 → 指挥官 → 精英 → 霸主）。

物种在以下方面各异：
- **伤害类型** —— 它们造成的伤害种类
- **抗性与弱点** —— 有些弱于等离子，有些弱于爆炸物
- **武器偏好** —— 低军衔用手枪，高军衔用重型武器
- **形态** —— 影响属性与抗性的身体构造

这意味着**每局游戏都面临不同的异形威胁**。一局可能遇到以灵能为主的物种，
另一局可能是弱于爆炸物的近战掠食者。

### 形态

形态决定异形的身体形态。关键因素：

**肢体：**
- 手臂（0–6）：手臂越少 = 命中率越差，手臂越多 = 稳定性更好或可双手持械
- 腿（0–8）：腿越多 = 越快但目标更大；零腿 = 悬浮，更难命中

**身体类型及其抗性：**
- **血肉之躯：** +动能抗性，-爆炸弱点
- **基于硅的晶体组织：** +激光/+等离子抗性，-爆炸弱点，反光
- **半实体的气态形态：** 免疫动能，弱于等离子，可穿墙
- **致密的矿物晶体结构：** 全方位良好抗性，极弱于爆炸物，死亡时碎裂
- **不断变化的无定形躯体：** +灵能抗性，每回合恢复 HP
- **精密锻造的机械组件：** 免疫灵能，+等离子抗性，-激光弱点，自爆
- **有机组织与合成装甲的融合体：** 均衡抗性，治疗相邻异形
- **纳米级机器集群：** +动能抗性，死亡时可复活

**感官：**
- **眼睛：** 影响命中率 —— 多光谱忽略烟雾/黑暗
- **听觉：** 回声定位在近距离透过烟雾侦测单位
- **热感：** 近距离无视掩体侦测生物单位
- **灵觉：** 提升灵能，侦测被精神控制的人类
- **化学感应：** 对受伤目标的命中加成

### 知识等级

随着您遭遇异形，情报会改善：

| 等级 | 您学到的内容 |
|-------|----------------|
| 未知 | 名称显示为“???” |
| 已发现 | 名称与图标揭露 |
| 已击杀 | 属性与抗性揭露 |
| 已解剖 | 完整的背景知识与详细弱点 |

### 异形 AI

异形会巡逻，直到发现人类，然后攻击。行为包括：
- **搜索** —— 朝最后已知位置移动数个回合
- **逃跑** —— 受伤严重且勇气低时逃离
- **适应** —— 异形会跨任务研究您的战术。
  远程狙击？它们会冲向您。用手榴弹？它们会散开。
  频繁包抄？它们会布置压制者。

### 装备升级

随着战役推进，异形获得更好的装备：
- **早期月份：** 等离子手枪、基础护甲
- **战役中期：** 等离子步枪、重型等离子、异形火炮
- **战役后期：** 顶级异形武器与护甲

### 俘虏异形

使用**电击棒**（近战，制造需 $2K）将异形击晕。
若眩晕伤害超过其 HP，它们会陷入昏迷，并可在任务后收集 ——
前提是您有带空闲容量的外星人收容所。

被俘异形可从研究界面审讯（`I` 键）：
- 审讯可立即完成一项正在进行的尸体解剖
- 或给予当前研究进度加成
- 需要至少一个实验室

### 属性成长

士兵通过战斗中的**逐动作经验**提升：
- **射击** → 提升精准
- **反应** → 提升反应
- **近战** → 提升力量
- **勇气** → 提升勇气（来自抵抗恐慌）
- **灵能技能** → 提升灵能技能与灵能力量

每次任务后，累积的经验转化为属性增益。获得经验的士兵还会朝其
HP、行动点和力量的上限获得通用“光环”成长。上限约为：行动点 80、
HP 60、精准 120、反应 100、勇气 100、力量 70、灵能 100。

### 疲劳与伤口

- **受伤士兵**在治愈前无法部署（每天恢复 2 HP）
- **疲劳：** 战斗导致 1–5 天疲劳
- 治疗设施与宿舍可加速恢复

### 致命伤

在战斗中，命中可能导致致命伤与流血。流血每回合消耗 HP —— 快用医疗包处理。
存活下来的伤口在任务后变为恢复天数。

### 士气

士兵每回合恢复士气。低士气可能触发恐慌（跳过回合）。
抵抗恐慌可建立勇气经验。

## 技巧与策略

### 早期游戏

1. 首先研究**外星合金** —— 它解锁激光武器与护甲。
2. 建造第二座**雷达** —— 更多探测，更多月度资金。
3. 招募 2–4 名额外士兵以填满您的小队。
4. 使用**武器设计室**创建自定义步枪 —— 借助合适的部件，您可以造出
   比量产型号更好的枪。
5. 不要忽视尸体解剖 —— 某些武器科技需要它们。
6. 在**飞机设计室**中自定义您的截击机 —— 均衡的设计
   （中等长度/翼展、2 引擎、刺鳐导弹）优于默认截击机。

### 战斗

- **利用掩体** —— 墙（80%）> 岩石（70%）> 树（60%）> 灌木（40%）
- **开火前蹲下**以获得更好命中率与伤害减免
- **手榴弹**绕过掩体并摧毁墙 —— 完美对付 entrenched 敌人
- **了解异形抗性** —— 首次击杀后查看图鉴
- **不要过度延伸** —— 当您在异形视线内移动时，它们会进行反应射击
- **保留一名医疗兵** —— 一名带医疗包的士兵可以救命
- **管理负重** —— 不要用沉重装备过度装载士兵

### 经济

- 出售多余的异形尸体与战利品换取现金
- 月度薪资累积 —— 在名册与收入之间取得平衡
- 委员会任务支付 $100K 额外奖金 —— 优先处理它们
- 早期制造物品出售以获利

### 研究路线

合金 → 激光武器 → 个人护甲 → 尸体解剖 → 埃勒里姆 → 等离子武器

中期：中型服、重型等离子。
后期：动力/飞行服、精神控制。

### 基地建造

- 雷达能自给自足（每个 +$50K/月）
- 尽早建造仓库 —— 您会很快装满
- 外星人收容所是活体俘虏与审讯加成的必要条件
- 相邻设施互相增益 —— 规划您的基地布局
- 若想要灵能能力，建造一座灵能实验室
