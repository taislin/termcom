# Быстрый старт termcom

ASCII-демейк X-COM для вашего терминала. Командуйте обороной человечества от вторжения пришельцев.

## Запуск

### Терминальная версия

```bash
go run ./cmd/termcom      # или: make run
```

### Версия браузера (WASM)

Сервер бэкенда не требуется — запускается напрямую в браузере.

```bash
# Сборка и запуск
cd cmd/termcom_wasm
GOOS=js GOARCH=wasm go build -o ../../web_wasm/termcom.wasm .
cd ../../web_wasm
python -m http.server 8080
# Открыть http://localhost:8080
```

Или используйте скрипт сборки: `./scripts/build_wasm.sh`

## Игровой цикл

1. **Геосфера** -- НЛО летят к городам. Обнаружьте и перехватите их.
2. **Перехват** -- Запустите истребители (L) или авторешите (A), чтобы сбить НЛО.
3. **Бой** -- Развернитесь на места крушений (R). Войдите в тактический бой.
4. **База** -- Исследуйте технологии пришельцев, производите снаряжение, нанимайте/экипируйте солдат.
5. **Повтор** -- Выиграйте 10 боёв, затем штурмуйте Сидонию, чтобы спасти Землю.

Проигрыш, если Активность пришельцев достигнет 100%.

## Основные клавиши (Геосфера)

| Клавиша | Действие |
|-----|--------|
| Space | Пауза |
| 1-4 | Скорость времени |
| L | Запустить перехватчик |
| A | Авторешение НЛО |
| M | Ответить на миссию |
| R | Отправить транспорт к крушению |
| B | Открыть базу |
| F5/F9 | Сохранить/Загрузить |
| Q | Выход |

## Основные клавиши (Боевая позиция)

| Клавиша | Действие |
|-----|--------|
| Arrow/WASD | Движение курсора |
| Space/Enter | Выбрать/Подтвердить |
| F | Огонь из оружия |
| R | Перезарядка |
| Q | Сменить солдата |
| E | Конец хода |
| C | Присесть |
| Esc | Отмена |

## Быстрая стратегия

- **Ранние этапы:** Нанимайте солдат, исследуйте Сплавы пришельцев, стройте Лабораторию + Мастерскую
- **Середина:** Кастомное лазерное оружие (Конструктор оружия) → Личная броня, расширяйте базы
- **Поздняя игра:** Кастомное плазменное оружие, Силовой/Летающий костюмы, пси-тренировка
- Всегда экипируйте солдат перед боем. Раненые лечат 2 ОЗ/день.
- Проектируйте кастомные перехватчики в Конструкторе самолётов — ракеты Стингрей + броня Лёгкий сплав — надёжное начало.
- Продавайте лишние артефакты пришельцев за наличные. Объекты Радара повышают финансирование.

## Победа

Выиграйте 10 наземных боёв, чтобы открыть финальную миссию Сидонии. Уничтожьте Сидонию, чтобы победить.

Полное руководство см. в [manual.ru.md](manual.ru.md).
