# termcom 快速上手

一款为您的终端打造的 ASCII X-COM 重制版。指挥人类抵御外星入侵的防御力量。

## 运行

```bash
go run ./cmd/termcom      # 或：make run
```

## 游戏循环

1. **地球仪表盘** -- UFO 飞向城市。侦测并拦截它们。
2. **拦截** -- 发射战斗机（L）或自动决斗（A）将其击落。
3. **战斗** -- 部署至坠毁地点（R）。进入战术战斗。
4. **基地** -- 研究外星科技、制造装备、招募/装备士兵。
5. **重复** -- 赢得 10 场战斗，然后突击赛多尼亚以拯救地球。

若外星活动达到 100% 则失败。

## 必备按键（地球仪表盘）

| 按键 | 操作 |
|-----|--------|
| Space | 暂停 |
| 1-4 | 时间速度 |
| L | 发射截击机 |
| A | 自动决斗 UFO |
| M | 响应任务 |
| R | 派遣运输机至坠毁点 |
| B | 打开基地 |
| F5/F9 | 保存/读取 |
| Q | 退出 |

## 必备按键（战场画面）

| 按键 | 操作 |
|-----|--------|
| 方向键/WASD | 移动光标 |
| Space/Enter | 选择/确认 |
| F | 开火 |
| R | 换弹 |
| Q | 切换士兵 |
| E | 结束回合 |
| C | 蹲下 |
| Esc | 取消 |

## 快速策略

- **早期：** 招募士兵、研究外星合金、建造实验室 + 车间
- **中期：** 自定义激光武器（武器设计室）→ 个人护甲，扩张基地
- **后期：** 自定义等离子武器、动力/飞行服、灵能训练
- 战斗前务必为士兵装备。受伤者每天恢复 2 HP。
- 在飞机设计室中设计自定义截击机 —— 刺鳐导弹 + 轻合金护甲是个不错的起点。
- 出售多余的外星 artifacts 换取现金。雷达设施提升资金。

## 胜利

赢得 10 场地面战斗以解锁赛多尼亚最终任务。摧毁赛多尼亚即获胜。
